## Installation
- ``composer install``
-  ``npm install``
-  ``npm run dev``

create a .env file and copy the content of .env.example
-  ``php artisan key:generate``
-  ``php artisan migrate``
- ``php artisan db:seed``
- ``php artisan serve``
- ``php artisan storage:link``
